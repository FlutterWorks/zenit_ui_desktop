# Color Scheme

Zenit UI has a few color roles:

## Primary

Primary colors are the most prominent colors displayed across an app, and are typically used for branding
or a user's theme choices. Primary colors may be vibrant.

## Secondary

Zenit's secondary color is a middle shade of gray. It is used on buttons, switches, and other components.

## Surface

A surface color is used by various secondary surfaces throughout the app. It should be a shade somewhere
between Secondary and Background. This may be used as a background color in place of acrylic when it isn't
available, such as in a mobile app, or where it doesn't make sense.

## Background

The color used in the furthest solid background of your app. On dahliaOS, this is secondary to the acrylic
background, which shows the desktop behind it.

# Theming in Zenit for Web (Web Components)

Zenit's low-level Web Components package (`@zenit-ui/web-components` or `zenit-ui`) uses CSS variables to manage its colors.

| Variable name                       | Default value                     | Description                                                                                                                                                                                                                                                            |
| ----------------------------------- | --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--zenit-theme-primary`             | `#00b26e`                         | The primary color, used for positive actions and branded accents around an app.                                                                                                                                                                                        |
| `--zenit-theme-on-primary`          | `#000000`                         | A color that contrasts with the primary color (preferably pure white or black).                                                                                                                                                                                        |
| `--zenit-theme-secondary`           | Light: `#e6e6e6`; Dark: `#262626` | A secondary color used for some interactive backgrounds (i.e. buttons and text fields). It is somewhat visible against the background.                                                                                                                                 |
| `--zenit-theme-on-secondary`        | Light: `#000000`; Dark: `#ffffff` | A color that contrasts with the secondary color.                                                                                                                                                                                                                       |
| `--zenit-theme-secondary-on-widget` | Light: `#535353`; Dark: `#FAFAFA` | A color used for contrast with the surface or secondary, but not used for text. This is used by the switch when it is turned off.                                                                                                                                      |
| `--zenit-theme-surface`             | Light: `#f7f7f7`; Dark: `#212121` | A color used for a non-interactive surface placed on top of a background, typically used by an app-bar or card. It should be closer to gray than the background color. The sidebar on web apps should use this color, but where possible, acrylic can be used instead. |
| `--zenit-theme-on-surface`          | Light: `#000000`; Dark: `#ffffff` | A color that contrasts with the surface color. Usually the same as `--zenit-theme-on-background`.                                                                                                                                                                      |
| `--zenit-theme-background`          | Light: `#ffffff`; Dark: `#121212` | The color of the solid background that covers the empty spaces behind everything in an app. This is used for non-acrylic areas in dahliaOS, but in most web applications it's going to be the primary background.                                                      |
| `--zenit-theme-on-background`       | Light: `#000000`; Dark: `#ffffff` | A color that contrasts with the background color.                                                                                                                                                                                                                      |
